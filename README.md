# astrbot_plugin_klbq_wiki

卡拉彼丘 Wiki 查询插件。

作者：凌溪

## 功能

- 使用命令 `/卡拉彼丘 角色名/武器` 查询卡拉彼丘 Biligame Wiki
- 优先提取页面结构化信息，并渲染成图片卡片
- 提取不到结构化内容时，自动回退到页面简介
- 支持别名输入，内置全部角色的常用简称、社区昵称和常见错字，英文别名不区分大小写
- 支持 `/卡拉彼丘 角色武器` 或 `/卡拉彼丘 角色的武器` 动态查询对应武器
- 支持配置是否发送详情链接
- 返回角色、武器的常用字段，如分距离部位伤害、基础伤害、部位系数、射速、瞄准速度、散射、后坐力、移速、弹匣容量、换弹/拉栓/蓄力时间等

## 用法

```text
/卡拉彼丘 心夏
/卡拉彼丘 空境
/卡拉彼丘 汐
/klbq 心夏
```

## 安装

1. 将插件目录放到 AstrBot 的 `plugin` 目录下
2. 安装依赖：

```bash
pip install -r requirements.txt
```

3. 重启 AstrBot

## 配置

插件支持以下配置项：

- `render_image`：是否将结果渲染为图片卡片
- `send_detail_link`：图片后是否再发送详情链接
- `image_timeout`：图片渲染超时时间，默认 8 秒
- `text_fallback`：图片渲染失败或超时时是否发送文字结果
- `grid_columns`：图片每行显示的资料格子数，支持 1-4
- `card_width`：卡片最小宽度，支持 420-1200 像素
- `custom_aliases`：自定义别名映射，每行格式为 `别名=页面标题`

示例：

```text
心夏老师=心夏
空境武器=空境
```

## 说明

插件会访问：

- https://wiki.biligame.com/klbq/%E9%A6%96%E9%A1%B5
- https://wiki.biligame.com/klbq/api.php

如果查询到的是角色页面，插件会尝试从页面信息框中提取字段并按固定模板输出。
如果查询到的是心夏，会额外提示可继续查询空境。
